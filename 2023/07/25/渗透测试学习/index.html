    4. 地址信息

      4.1 内网中捕获到一个以太帧，源MAC地址为：98-CA-33-02-27-B5；目的IP地址为：202.193.64.34，回答问题：该用户使用的什么品牌的设备，访问的是什么网站？并附截图。

      IP查询网站:IP/IPv6查询，服务器地址查询 - 站长工具 (chinaz.com)

      该用户使用的是Apple的电子设备，访问桂林电子科技大学的网站

      4.2 访问https://whatismyipaddress.com得到MyIP信息，利用ipconfig(Windows)或ifconfig(Linux)查看本机IP地址，两者值相同吗？如果不相同的话，说明原因。

      两者的值并不相同，因为在网站上查到的是本机在公网上的地址，这个IP地址是唯一的，在本机上用ipconfig命令查到的地址是在子网内分配的地址，两个地址的含义不同。

    5. NMAP使用（两位同学配合完成）

      5.1 利用NMAP（攻击方）扫描发现Metasploitable2（靶机，需下载虚拟机镜像）IP地址，同时Wireshark捕获报文，分析NMAP扫描发现该主机的工作原理，即报文交互情况；

      扫描靶机端口开放情况。并附截图。说明其中四个端口（知名端口）提供的服务，查阅资料，简要说明该服务的功能。

      ①　 ① ftp：服务器的上传 和下载，Internet上的控制文件的双向传输。同时，它也是一个应用程序（Application）。用户可以通过它把自己的PC机与世界各地所有运行FTP协议的服务器相连，访问服务器上的大量程序和信息。实现各种操作系统之间的文件交流，建立一个统一的文件传输协议。

      ② Ssh：SSH是专为远程登录会话和其他网络服务提供的安全性协议。利用SSH协议可以有效的防止远程管理过程中的信息泄露问题。

      ③ Telent： Telnet提供远程登录功能，使得用户在本地主机上运行Telnet客户端，就可登录到远端的Telnet服务器. 在本地输入的命令可以在服务器上运行，服务器把结果返回到本地，如同直接在服务器控制台上操作. 这样就可以在本地远程操作和控制服务器。

      ④ Smtp：SMTP是一种提供可靠且有效电子邮件传输的协议。是建立在FTP文件传输服务上的一种邮件服务，主要用于传输系统之间的邮件信息并提供来信有关的通知。

      在wireshark抓包信息如下，可以看到本机地址在网段内发送广播包，查看是否有存活主机，如果有arp回应就是该ip地址有存活主机。

      找到存活主机之后再用TPC协议查看端口情况，如果收到ACK+RST,则端口开放，即经过三次握手成功建立tcp

      5.2 利用NMAP扫描Metasploitable2的操作系统类型，并附截图。

      命令:nmap -O 被扫描主机ip

      （需要注意的是这里的O是大写，再nmap中严格区分大小写）

      操作系统类型为linux 2.6.x

      5.3 利用NMAP穷举 Metasploitable2上dvwa的登录账号和密码。

      利用nmap中的脚本爆破登录页面，先找到登录授权页面，可以看到利用的是80端口，但是被扫描的主机80端口没有打开，利用wireshark抓到的数据也显示没有经过三次握手，检查了被扫描主机确实是192.168.43.212，排除连接方式的错误，然后只能尝试重启被扫描的metasploitable，才打开了80端口。

      命令:nmap -p80 --script=http-form-brute -script-args=http-form-brute.path=/dvwa/login.php 192.168.43.212

      找到的账户名为admin，密码为password

      在利用wireshark抓包时要正确运用过滤语句，如果要查看tcp的响应情况，则需要看发送方是否有成功发送数据，接收方是否有回应，这种情况下要用到ip.addr==，以便查看。

      在上图可以看到192.168.43.177的80端口是没有开的。

      上图的192.168.43.92的80端口是打开的。

      5.4 查阅资料，永恒之蓝-WannaCry蠕虫利用漏洞的相关信息。

      ​ 这种攻击应该是利用了微软系统的一个漏洞。该漏洞其实最早是美国国安局发现的，他们还给漏洞取名为EternalBlue(永恒之蓝)。Microsoft 服务器消息块 (SMB) 协议是 Microsoft Windows 中使用的一项 Microsoft 网络文件共享协议。在大部分 windows 系统中都是默认开启的，用于在计算机间共享文件、打印机等。Windows SMB远程提权漏洞，NSA 泄露工具EternalBlue利用 SMB可以攻击开放了445 端口的 Windows 系统并提升至系统权限。攻击者与TCP协议的445端口建立请求连接，获得指定局域网内的各种共享信息，并对文件施行加密等破坏性攻击。深信服安全云早在一个月前已更新微软SMB漏洞检测方案，并提供了安全应对方案
